# Customer Data Preprocessing

## Tasks Completed:
1. Data Cleaning
   - Filled missing values (median & mode)
   - Removed duplicates

2. Encoding
   - One-hot encoding for city
   - Label encoding for gender

3. Feature Scaling
   - Min-Max Scaling
   - Standardisation

## Tools Used:
- Pandas
- NumPy
- Scikit-learn

## Summary:
Prepared raw dataset for machine learning by cleaning, encoding, and scaling features.
